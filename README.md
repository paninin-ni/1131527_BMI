# 視窗（二）BMI計算機

### 專案簡介:
> 提供直觀的介面供使用者計算身體質量指數(BMI)，同時呈現理想體重。具備完善的輸入驗證機制，能針對非數字或負數輸入彈出錯誤提示，並以不同顏色標示狀態結果。

### 執行說明:
> 1. 輸入數據：在「身高(cm)」與「體重(kg)」欄位輸入數值
> 2. 點擊計算：按下「計算」按鈕後，程式會自動進行單位轉換與邏輯判斷
> 3. 查看結果：BMI 指數：顯示於文字框，並根據結果變更背景顏色（如：健康為綠色、肥胖為紅色）/ 理想體重：程式依據身高自動算出理想體重
